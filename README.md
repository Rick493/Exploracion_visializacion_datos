# Exploracion_visializacion_datos

Este database es sobre Chocolate Sales Dataset 2023 - 2024, lo hemos sacado de la pagina https://www.kaggle.com/datasets/ssssws/chocolate-sales-dataset-2023-2024?resource=download

Este conjunto de datos contiene datos de transacciones minoristas sintéticas para las ventas de chocolate en múltiples tiendas, clientes y productos durante los años 2023-2024.

El conjunto de datos está diseñado para el análisis de datos, el aprendizaje automático y los proyectos de pronóstico, simulando datos de ventas minoristas realistas que generalmente se encuentran en la industria de FMCG (Fast-Moving Consumer Goods).

Se puede utilizar para tareas como:

    Análisis de ventas
    Análisis de comportamiento del cliente
    Evaluación del rendimiento al por menor
    Predicción de series de tiempo
    Experimentos de aprendizaje automático
